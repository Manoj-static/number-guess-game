# 🎯 Number Guessing Game

A modern and interactive **Number Guessing Game** built with **HTML, CSS, and JavaScript**. The game challenges players to guess a randomly generated number within a selected difficulty range while providing real-time hints, attempt tracking, and performance statistics. It features a clean responsive interface, dark mode, animated confetti celebration on winning, a live narrowing range, guess history, and local best-score storage to create an engaging user experience.

## ✨ Features

* 🎯 Random number generation
* 🎚️ Multiple difficulty levels (Easy, Medium, Hard)
* 💡 Real-time hints (Higher/Lower)
* 📉 Live narrowing range indicator
* 📜 Guess history tracking
* ⏱️ Timer to measure completion time
* 🔢 Attempt counter
* 🏆 Best score saved using Local Storage
* 🌙 Dark/Light mode toggle
* 🎉 Confetti animation on winning
* 📱 Fully responsive design
* ⚡ Built with pure HTML, CSS, and JavaScript (No frameworks)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Canvas API (Confetti Animation)
